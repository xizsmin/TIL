# Collection in Java?
- 여러 원소들을 담을 수 있는 자료구조
- 제일 기본: 배열

### 자바에서의 자료구조 유형
- **`List`** 순서 있는 목록
  - ArrayList, LinkedList, Vector, Stack
  - **ArrayList는 thread-safe** (*not like Vector*)

- **`Set`** 순서 없는 목록
- **`Queue`** FIFO
- **`Map`** key-value pair (k, v 둘다 존재해야 하고, k는 고유)
  - 데이터 추가한 시간 순서 != 저장되는 순서
  - HashMap, TreeMap
