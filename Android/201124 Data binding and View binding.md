## Data binding
- Android JetPack 구성요소(Architecture)
- 레이아웃에 정의한 UI 구성요소를 app의 로직 / 데이터 소스와 결합
- xml 파일에 데이터를 연결 -> app logic과 레이아웃 요소를 간편하게 binding
  - `findViewById` 없이도 xml에 정의한 view들을 생성
  - xml의 최상위 root는 `<layout>` tag
